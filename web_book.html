1. HTML พื้นฐาน
โครงสร้างพื้นฐาน HTML📄 structure.html 
<!-- โครงสร้างบังคับทุกไฟล์ -->
<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ชื่อหน้า</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <!-- เนื้อหาอยู่ใน body -->
    <script src="app.js"></script>  <!-- JS ต่อท้าย body -->
  </body>
</html>
แท็ก หน้าที่ ตัวอย่าง
<h1>–<h6> หัวข้อ (h1=ใหญ่สุด) <h1>ชื่อเว็บ</h1>
<p> ย่อหน้าข้อความ <p>ข้อความ</p>
<a> ลิงก์ → ต้องมี href <a href="url">คลิก</a>
<img> รูปภาพ → src, alt <img src="pic.jpg" alt="desc">
<div> กล่อง block-level จัดกลุ่มเนื้อหา
<span> inline container ไฮไลต์ข้อความ
<ul> <ol> <li> รายการ (กระสุน/เลข) เมนู, ลิสต์
<header> <footer> <nav> <main> <section> Semantic HTML โครงสร้างหน้าเว็บ

Attribute สำคัญ ⚠️

  • id — ระบุ element เดียว (ไม่ซ้ำ)
  • class — ใช้หลาย element ได้
  • src — ที่อยู่รูป/ไฟล์
  • href — ลิงก์ปลายทาง
  • alt — คำอธิบายรูป

จำให้ขึ้นใจ 🔥

  • #id = เลือกด้วย id
  • .class = เลือกด้วย class
  • แท็กที่ไม่มีปิด: <img> <input> <br> <hr> <meta> <link>

2. Form
💡
จุดสำคัญ: method="GET" ส่งข้อมูลใน URL (เห็นได้) · method="POST" ส่งใน body (ปลอดภัยกว่า)
โครงสร้าง Form
<form action="submit.php" method="post">

  <!-- label เชื่อมกับ input ด้วย for = id -->
  <label for="username">ชื่อผู้ใช้:</label>
  <input type="text" id="username" name="username" placeholder="กรอกชื่อ">

  <!-- radio: name เดียวกัน = กลุ่มเดียวกัน -->
  <input type="radio" name="gender" value="male" id="male">
  <label for="male">ชาย</label>

  <input type="radio" name="gender" value="female" id="female">
  <label for="female">หญิง</label>

  <!-- checkbox: แต่ละอันมี name-value เป็นตัวเอง -->
  <input type="checkbox" name="hobby" value="code" id="h1">
  <label for="h1">เขียนโค้ด</label>

  <button type="submit">ส่งข้อมูล</button>
</form>
type ลักษณะ
text กล่องข้อความทั่วไป
password ซ่อนตัวอักษร
email ตรวจรูปแบบอีเมลอัตโนมัติ
number กรอกตัวเลข
radio เลือกได้อันเดียวในกลุ่ม (name เดียวกัน)
checkbox ติ๊กได้หลายอัน
submit ปุ่มส่งฟอร์ม
reset ล้างข้อมูลในฟอร์ม
⚠️
จำไว้: for ใน <label> ต้องตรงกับ id ของ <input> เสมอ ไม่ใช่ name!

3. Table
โครงสร้าง Table + colspan + rowspan 
<table border="1">
  <thead>
    <tr>
      <!-- colspan: รวม 2 คอลัมน์ในแนวนอน -->
      <th colspan="2">ชื่อ-นามสกุล</th>
      <th>คะแนน</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <!-- rowspan: รวม 2 แถวในแนวตั้ง -->
      <td rowspan="2">สมชาย</td>
      <td>ใจดี</td>
      <td>90</td>
    </tr>
    <tr>
      <!-- ไม่ต้องใส่ td แรก เพราะถูก rowspan ครอบแล้ว -->
      <td>รักเรียน</td>
      <td>85</td>
    </tr>
  </tbody>
</table>

Tags ใน Table

  • <table> — ตาราง (ห่อทั้งหมด)
  • <thead> — ส่วนหัวตาราง
  • <tbody> — ส่วนเนื้อหา
  • <tr> — แถว (table row)
  • <th> — เซลล์หัว (bold+center)
  • <td> — เซลล์ข้อมูล

colspan vs rowspan

  • colspan="2" → รวม 2 คอลัมน์ (แนวนอน ↔)
  • rowspan="3" → รวม 3 แถว (แนวตั้ง ↕)
  • เมื่อ rowspan ครอบแถวถัดไป → แถวถัดไปมี td น้อยลง 1

4. CSS พื้นฐาน
Selector + Properties สำคัญ 
/* Selector แบบต่าง ๆ */
h1          { color: red; }           /* tag selector */
.card       { padding: 1rem; }       /* class selector */
#header     { background: blue; }    /* id selector */
.card.box   { border: 1px solid; }  /* มี class ทั้งสอง */
.nav a      { color: white; }        /* a ที่อยู่ใน .nav */

/* Pseudo-class & Pseudo-element */
a:hover        { color: orange; }     /* เมื่อ hover */
.btn::before   { content: "★"; }    /* เพิ่ม content ก่อน element */

/* Transition */
.btn {
  background: blue;
  transition: background 0.3s ease;  /* animate เมื่อค่าเปลี่ยน */
}
.btn:hover { background: darkblue; }
Property ความหมาย ตัวอย่าง
display: block กินเต็มแถว div, p, h1
display: inline อยู่ในบรรทัด ไม่มีความกว้าง/สูง span, a, strong
display: inline-block inline แต่กำหนดขนาดได้ ปุ่ม, รูปภาพ
display: flex Flexbox layout จัดเรียงแนวนอน/ตั้ง
display: grid Grid layout จัดตาราง

หน่วย CSS ที่ต้องรู้

  • px — pixel คงที่ ไม่เปลี่ยนตามขนาดหน้าจอ
  • rem — สัมพันธ์กับ font-size ของ root (html) · 1rem = 16px โดยทั่วไป
  • % — สัมพัทธ์กับ parent element
  • vw / vh — % ของความกว้าง/สูงของ viewport

5. Flexbox & Grid
Flexbox
.container {
  display: flex;
  flex-direction: row;          /* row | column */
  justify-content: space-between; /* จัด main axis */
  align-items: center;           /* จัด cross axis */
  flex-wrap: wrap;               /* ตัดบรรทัดถ้าล้น */
  gap: 1rem;                      /* ช่องว่างระหว่าง items */
}

.item {
  flex: 1;   /* ขยายเต็มพื้นที่ที่เหลือ (เท่า ๆ กัน) */
}
📌
justify-content ค่าสำคัญ: flex-start | flex-end | center | space-between | space-around
CSS Grid
.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* 3 คอลัมน์เท่ากัน */
  grid-template-columns: repeat(3, 1fr); /* เขียนย่อ */
  gap: 1rem;
}

7. Bootstrap
Bootstrap Grid System
<!-- Grid 12 คอลัมน์ -->
<div class="container">          <!-- จำกัดความกว้าง, center -->
  <div class="row">               <!-- แถว flex -->
    <div class="col-6">ครึ่งซ้าย</div>   <!-- 6/12 = 50% -->
    <div class="col-6">ครึ่งขวา</div>
  </div>
  <div class="row">
    <div class="col-md-4">3 คอลัมน์</div>  <!-- ≥md=4/12, <md=12/12 -->
    <div class="col-md-4">3 คอลัมน์</div>
    <div class="col-md-4">3 คอลัมน์</div>
  </div>
</div>
Class ความหมาย
container กล่องกลาง มีความกว้างจำกัด
container-fluid กว้างเต็ม 100%
row แถวใน grid (ใช้ flexbox)
col-6 6/12 = 50% ทุกขนาดหน้าจอ
col-md-4 4/12 เมื่อ ≥768px, เต็มเมื่อน้อยกว่า
d-flex / d-none display:flex / display:none
mt-3 / p-2 margin-top:1rem / padding:.5rem (1=.25, 2=.5, 3=1, 4=1.5, 5=3rem)
text-center / text-end จัดข้อความ
btn btn-primary ปุ่มสีน้ำเงิน Bootstrap
bg-primary / text-danger สีพื้นหลัง / สีข้อความ
🎨
สี Bootstrap: primary (น้ำเงิน) · secondary (เทา) · success (เขียว) · danger (แดง) · warning (เหลือง) · info (ฟ้า) · light · dark

8. JavaScript (Workshop)
getElementById + classList.toggle 
// getElementById: เลือก element ด้วย id
const box = document.getElementById('myBox');

// classList.toggle: เพิ่ม class ถ้าไม่มี / ลบถ้ามีอยู่
box.classList.toggle('active');

// ตัวอย่าง: ปุ่มเปิด/ปิด dark mode
<button onclick="toggleDark()">Toggle Dark</button>
<div id="page">...</div>

function toggleDark() {
  document.getElementById('page')
          .classList.toggle('dark-mode');
}

classList methods

  • .toggle('x') → เพิ่ม/ลบสลับกัน
  • .add('x') → เพิ่ม class
  • .remove('x') → ลบ class
  • .contains('x') → ตรวจว่ามีหรือไม่

DOM Selection

  • getElementById('id')
  • querySelector('.class')
  • querySelectorAll('tag')
